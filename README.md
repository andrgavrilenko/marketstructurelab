# MarketStructureLab

Landing page for MarketStructureLab — a TradingView indicator studio focused on market structure, liquidity, and volume analysis.

## Setup

1. Clone the repo
2. Copy `config.example.js` to `config.js`
3. Replace `YOUR_FORM_ID` with your [Formspree](https://formspree.io) form ID
4. Open `index.html` in a browser or deploy to any static hosting

## Stack

Single static HTML page — no build tools, no dependencies. Fonts loaded from Google Fonts CDN.

## Deploy

Works with any static hosting: GitHub Pages, Netlify, Vercel, or just open the file directly.
